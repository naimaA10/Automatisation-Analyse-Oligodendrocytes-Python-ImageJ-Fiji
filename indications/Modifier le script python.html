Si nous avons une liste contenant des noms de fichiers au format csv pour la parcourir il faut utiliser une boucle for. Un exemple est prsent dans la cellule suivante. La ligne de code for csv in liste permet de parcourir les fichiers un par un donc "img1_A01.csv" puis "img2_A01.csv", "img1_A02.csv" et enfin "img2_A02.csv". Pour chaque fichier affich un message avec print le message affich est Nouveau fichier suivi du nom du fichier csv et du puits associ. Ici csv correspond au nom du fichier exemple img2_A01.csv qui est une chane de caractres, il est possible de rcuprer une sous-chaine de caractre exemple :

  • csv correspond img2_A01.csv
  • csv[0:4] correspond la sous chaine de caractre 0 4 exclus donc rcuprer les caractres img. Le caractre 0 correspond i le caractre 2 m et le caractre 3 g.
  • csv[-1] correspond au premier caractre en partant de la fin (car il y a le symbole -) donc il s'agit de v
  • csv[-7:-4] correspond A01 car il part du 7e caractre partir de la fin donc A jusqu'au 4e caractre en partant de la fin exclue donc "." n'est pas pris en compte. Cette mthode est ensuite utilise dans le script pour rcuprer le nom du puits qui est en train d'tre analys et concatner les fichiers d'un mme puits.
